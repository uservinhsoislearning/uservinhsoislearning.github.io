# Vinh's Data & AI Blog

This repository contains the source code for my personal website and blog, hosted on GitHub Pages. 

## Overview
The site serves as a portfolio to showcase my projects in Machine Learning, Big Data, and software engineering, alongside my technical blog posts. It also links to an external drive for sharing large datasets, project files, and models.

## Project Structure
* `index.html`: The main blog feed and external storage links.
* `about.html`: My professional background and technical skills.
* `style.css`: Custom, responsive styling for the site.
